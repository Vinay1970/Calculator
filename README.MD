# Scientific Calculator with Financial and Date Tools

A feature-rich calculator application built with Python and Tkinter, offering scientific, financial, and date/time calculation capabilities.

## Features
* **Scientific Calculator**: Supports basic arithmetic, trigonometry, logarithms, exponentiation, and more.
* **Currency Converter**: Live exchange rates and static conversion options.
* **Financial Tools**: Simple interest, compound interest, and loan payment calculations.
* **Date/Time Tools**: Date differences and date arithmetic.
* **History**: Tracks previous calculations.
* **Memory Functions**: M+, M-, MR, MC.

## Requirements
* Python 3.x
* Tkinter (usually included with Python)
* `simpleeval` library (install with `pip install simpleeval`)

## Usage
1. Run the script: `python calculator.py`
2. Use the calculator interface to perform calculations.
